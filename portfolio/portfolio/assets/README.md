## 📸 Adding Images

### Profile Photo
- Place your photo at: `assets/images/profile/photo.jpg`
- Recommended size: 400×500px or any portrait ratio
- Supported formats: .jpg, .jpeg, .png, .webp

### Project Screenshots
Place project screenshots in `assets/images/projects/` with these exact filenames:
- `emotion-ai.jpg` — Screenshot or illustration for Emotion AI project
- `inventory.jpg`  — Screenshot for Inventory Management System
- `medtrack.jpg`   — Screenshot for MedTrack app
- `weather.jpg`    — Screenshot for Weather App
- `portfolio.jpg`  — Screenshot for Portfolio website

### Tips for Good Project Screenshots
- Recommended size: 800×450px (16:9 ratio)
- Use screenshots from your actual projects
- Or use tools like Figma/Canva to create attractive cover images
- All images gracefully fall back to icon placeholders if not found
